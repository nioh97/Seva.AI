This version merges backend + frontend into one Next.js project.

## Setup

1. Extract folder
2. Install dependencies:

```bash
npm install
```

3. Run in development:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production Build

```bash
npm run build
npm start
```

The chat API is available at `/api/chat`.
